Project Description

The purpose of my project is to build a mobile application controlling NAO Humanoid Robot that can provide multiple modules to assist autistic children in a way comfortably socializing with others step by step. This project will also develop an intelligent web interface dashboard to collect applicable data extracted from the robot and mobile human-robot interaction software for therapists and parents, which can help them analyze the condition of patients/children. My responsibility is to design its architecture, database, mobile, and web interface, develop core modules, and deal with human-robot interaction.

Main Points

Designed architecture of whole platform by software engineering principle and developed different functionality components by design pattern for better scalable and improved developing efficiency.

Integrated Spring Boot, Mybatis, IBM Watson, Naoqi as back end core framework. Developed mobile interface(Swift), web interface(React JS) for front end. Implemented cross-platform calls and end-to-end data interaction.

Designed human-robot interaction module for autistic children, using a series of components such as sensor, speech, vision and motion of the robot.
